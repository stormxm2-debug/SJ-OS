# SJ OS 모바일 접속 안내 (직원용)

[SJ OS 모바일 접속 안내]

1. 전달받은 주소로 접속합니다. (예: `https://<사이트>.netlify.app`)
2. 로그인합니다. (관리자에게 받은 이메일/비밀번호)
3. 홈 화면을 확인합니다. (오늘 출근/일정/공지 등)
4. 출퇴근 / 고객 / 상담 / 일정을 사용합니다.
   - 하단 탭: 홈 · 출퇴근 · 고객 · 일정 · 더보기
   - 더보기: 상담기록 · 실적관리 · 공지사항 · 자비스 · 로그아웃
5. 자주 쓰면 **홈 화면에 추가**합니다.
   - iPhone: 공유 → 홈 화면에 추가
   - Android: 메뉴 → 앱 설치 / 홈 화면에 추가
6. 오류 발생 시 **화면 캡처** 후 관리자에게 전달합니다.

## 참고

- 개발/관리자 기능은 모바일에서 보이지 않습니다. (데스크톱 앱에서만 사용)
- 데이터는 로그인한 본인 권한에 맞게 표시됩니다. (FC: 본인 / 팀장: 팀 / 대표: 전체)
- 로그아웃은 더보기 → 로그아웃에서 할 수 있습니다.
