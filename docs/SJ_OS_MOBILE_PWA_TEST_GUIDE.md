# SJ OS 모바일 PWA 테스트 가이드

모바일 스태프 화면을 확인하는 방법입니다. (데스크톱 Electron 앱은 그대로 유지됩니다.)

## 1. 개발 실행

```
npm run dev
```

- Electron 창이 열립니다. 데스크톱 폭에서는 기존 데스크톱 화면이 그대로 보입니다.

## 2. 모바일 화면 확인 (개발자 도구)

1. 앱 창에서 개발자 도구를 엽니다 (보기 → 개발자 도구 또는 Ctrl+Shift+I).
2. 디바이스 툴바(모바일 아이콘)를 켭니다.
3. 폭을 **390px** 정도로 설정합니다.
4. 하단 탭바(홈·출퇴근·고객·일정·더보기)가 나타나는지 확인합니다.
5. 개발/릴리즈/배포 메뉴가 **보이지 않는지** 확인합니다.

> 웹서버로 실제 휴대폰에서 여는 방식(호스팅)은 다음 단계입니다. 이번 스프린트는 모바일 UI
> 기반만 추가했습니다.

## 3. 로그인 테스트

- Supabase 미설정 시: 로컬 데모 로그인(대표/팀장/FC)으로 진입합니다.
- Supabase 설정 시: 이메일/비밀번호 로그인 후 프로필 권한으로 진입합니다.

## 4. 기능 테스트

- **출퇴근**: 출근하기/퇴근하기 → 오늘 기록/상태 확인.
- **고객 등록**: 고객 탭 → 고객 등록 → 저장.
- **상담기록**: 더보기 → 상담기록 → 고객 선택 후 작성.
- **일정**: 일정 탭 → 일정 등록.
- **실적**: 더보기 → 실적관리 요약 확인.
- **공지사항**: 더보기 → 공지사항.
- **자비스**: 더보기 → 자비스 또는 홈의 “자비스 도움받기”.

## 5. 홈 화면에 추가 (PWA)

- 실제 호스팅 후 모바일 브라우저에서 접속하면 “홈 화면에 추가”로 설치할 수 있습니다.
- 설치 시 앱 이름은 **SJ OS**, 세로 화면(standalone)으로 실행됩니다.
- 이번 단계에서는 매니페스트만 추가되었고, 오프라인 캐시(서비스 워커)는 민감정보 보호를
  위해 넣지 않았습니다.

## 6. 문제 보고

- 화면 캡처 + 사용 폭(px) + 재현 단계를 담당자에게 전달해주세요.
- 클릭이 안 되거나 흰 화면이 나오면 앱을 재시작 후 다시 시도하고, 반복되면 보고해주세요.
